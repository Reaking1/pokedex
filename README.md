# 📖 Pokédex

*A modern Pokédex web application built with React, JavaScript, SCSS, and the Pokémon API.*

---

## 📖 Overview

Pokédex is a responsive web application that allows users to browse, search, and explore Pokémon using real-time data from the Pokémon API. Built with **React** and **JavaScript**, the application provides a clean and interactive interface for viewing Pokémon information, including images, types, abilities, and base statistics.

This project demonstrates modern frontend development practices, component-based architecture, API integration, and responsive design.

---

## ✨ Features

* 🔍 Search Pokémon by name
* 📖 Browse Pokémon collection
* 🖼️ Official Pokémon artwork
* ⚡ View Pokémon types
* 📊 Display base stats
* 💪 View abilities
* 📱 Fully responsive design
* 🔄 Real-time API data

## ⚙️ Requirements

Before running the project, install:

* 🟢 Node.js 22 LTS (or later)
* 📦 npm
* 💻 Visual Studio Code
* 🌐 Git

---

## 📥 Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/pokedex.git
```

---

### 2️⃣ Navigate to the Project

```bash
cd pokedex
```

---

### 3️⃣ Install Dependencies

```bash
npm install
```

---

## ▶️ Running the Project

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
